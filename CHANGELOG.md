# Changelog - Hermes Dashboard

## [1.2.0] - SaaS & Monetization Update (Hold 1 Release) - 2026-04-09
### Added
- **Public Landing Page**: Dedikeret salgsside for Hermes Cloud med features, testimonials og pricing grid.
- **Subscription Engine**: Nyt "Konto" tab i Settings med Stripe-checkout integration.
- **War Room Approvals**: Risiko-baseret farvekodning (Rød/Gul/Grøn) for agent-handlinger.
- **Neural Memory Graph**: Interaktiv visualisering af agentens kognitive netværk (Jonas, Rawan, Rendetalje).
- **Usage Tracking**: Backend infrastruktur til at tracke API-forbrug for Free/Pro planer.

### Changed
- **Onboarding 2.0**: Transformeret fra teknisk setup til en høj-konverterende feature-pitch.
- **Terminal UX**: Tilføjet blå "thought-stream" logs for bedre gennemsigtighed i AI-tænkning.

## [1.1.0] - Foundation Update
- Basic dashboard structure, terminal, and session management.
- Initial model selector.
